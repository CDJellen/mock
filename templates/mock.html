Simulation scenarios provide mock ability responses - per agent - during a running operation. Use scenarios to test planners, as well as see how an attacker may make decisions if they face specific defensive measures. Apply a scenario and run a batch of operations below.

A batch is a set of n-number of operations. Run a batch to test out different planners and understand how they compare against each other.
