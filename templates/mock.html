Mock

an environment to simulate operations

Simulation scenarios provide mock ability responses - per agent - during a running operation. Use scenarios to test planners, as well as see how an attacker may make decisions if they face specific defensive measures. Only one scenario can be applied at a time. Apply a scenario and run a mock trial below.


Trials


A trial is a set of n-number of operations. Run trials to test out different planners and understand how they compare against each other.
